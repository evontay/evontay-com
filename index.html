Open source data project for the Singapore startup community. Backend API built with Node.JS + Express. Frontend built with HTML, CSS, Javascript, Ajax, JQuery, Algolia. Collaborators - Geng Sng, Lew Li Min, Winston Mi.

A simple app that advises on the daily weather to decide whether it is a good day for biking. Used the 24-hour weather forecast API from data.gov.sg. Built with Ajax, Jquery, HTML, CSS.
